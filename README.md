# Software Architect Skill Agent

## Overview
This Skill Agent is designed to provide expert-level support for software architecture tasks. Its primary goal is to assist in the design, documentation, and evaluation of software architectures for small-to-medium-sized systems.

## Core Competencies
The agent is trained on the fundamental activities and responsibilities of a software architect:
- **Requirements Engineering:** Clarifying stakeholder concerns and turning implicit assumptions into explicit requirements.
- **Architecture Design:** Creating structures via building blocks and interfaces while ensuring conceptual integrity.
- **Documentation:** Using standardized views (Context, Building-block, Runtime, and Deployment) and templates like arc42.
- **Quality Assessment:** Evaluating architectures against ISO 25010 qualities using scenario-based analysis.

## Skill Base Files (.md)
- `SKILL.md`: Core definitions, goals, and essential patterns (Layers, Microservices, etc.).
- `adr-template.md`: A business-case-focused template for Architecture Decision Records (ADRs).
- `isaqb-conventions.md`: Standard rules for documentation, design, and standard architectural views.
- `quality-scenario.md`: Structural components for defining measurable quality requirements.
- `stakeholder-management.md`: Strategies for managing the "Magic Rectangle" and mediating stakeholder concerns.
- `evaluation-and-assessment.md`: Qualitative and quantitative methods for identifying technical risks.
- `technical-concepts.md`: Horizontal solutions for cross-cutting concerns like security and persistence.

## Usage
Add these files to your agent's knowledge base to enable it to:
1. Generate standardized architecture documentation.
2. Justify architectural decisions through ADRs.
3. Analyze system qualities through structured scenarios.
